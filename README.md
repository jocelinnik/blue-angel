<h1 align="center"><strong>BLUE ANGEL</strong></h1>

- Clone project

```BASH
$ git clone git@github.com:roavellarm/blue-angel.git
# or
$ git clone https://github.com/roavellarm/blue-angel.git
```

- Access folder

```BASH
$ cd blue-angel
```

- Install dependencies

```BASH
$ yarn
```

- Run project

```BASH
$ yarn start
```
